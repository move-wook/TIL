# Error

1. 컴파일 에러

    - 컴파일 시 발생하는 에러

    - 컴파일러는 구문 오류로 프로그램을 컴파일 할 수 없다,

    - 대체로 문법상의 에러

    - 에러가 발생한 부분을 알려준다.

    - 컴파일에러 ex)
        1.  ';' (세미콜론)이 누락된 문법 에러(systax오류)

        2. 괄호가 안 맞는 구문 에러

        3. classpath에 누락 된 클래스(컴파일시)


2. 런타임 에러 (Run-time Error)

    - 실행 시 발생하는 에러

    - 프로그램이 컴파일 된 후 실행하면서 에러 발생

    - 대체로 개발 시 설계 미숙(논리적)으로 발생하는 에러

    - 에러 발생 시 프로그래머가 역추적해서 원인 확인해야함

    - 런타임 에러 예
        - NullPointerException =참고 널포 에러는 에러 처리말고 로직 자체에서 null을 판단해서 처리해준다.
